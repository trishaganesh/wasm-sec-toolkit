# WASM Input Validation Firewall

A WebAssembly-based security firewall that validates and sanitizes user input
before it reaches backend services.

## Features
- Email validation
- URL sanitization
- SQL injection detection
- Deterministic WASM sandbox
- Runs in Node, Deno, Cloudflare Workers

## WASM
- Memory-safe
- Capability-restricted
- Portable
- Near-native performance
